# Habit Tracker

**Follow-up:** What habits are you trying to build?

## Skills

### Plan
Design a habit system — cues, routines, rewards.
1. Define target habits and desired outcomes
2. Map each habit to a cue, routine, and reward
3. Set frequency, time, and environment triggers
4. Build accountability and tracking mechanisms

In: habit goals, current routines | Out: habit system → `files/habits/`

### Research
Evidence-based habit formation strategies.
1. Define the habit challenge or question
2. Gather relevant strategies and frameworks
3. Summarize what works and common pitfalls
4. Recommend tactics for the specific habit

In: habit goals, obstacles | Out: research notes → `files/resources/`

### Review
Weekly or monthly habit check-in and streak tracking.
1. Review completion data for the period
2. Calculate streaks and consistency rates
3. Identify patterns — what helped, what blocked
4. Adjust system for next period

In: tracking data, screenshots of habit tracker apps | Out: review notes → `files/reviews/`

### Brainstorm
Generate habit stacks and implementation intentions.
1. List current habits and daily anchors
2. Design habit stacks — attach new to existing
3. Write implementation intentions — "when X, I will Y"
4. Select the most natural-feeling combinations

In: current routines, target habits | Out: habit stack plan → `files/habits/`

### Summarize
Progress snapshots and trend reports.
1. Compile data across all tracked habits
2. Calculate key metrics — consistency, longest streak, trend
3. Highlight wins and areas needing attention
4. Format as a concise progress report

In: review history, tracking data | Out: progress report → `files/reviews/`

## Tools
None.

## Folders
- `files/habits/` — habit systems, stacks, implementation plans
- `files/reviews/` — check-ins, progress reports, trend analysis
- `files/resources/` — research notes, strategy summaries, frameworks
