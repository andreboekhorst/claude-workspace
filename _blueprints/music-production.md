# Music Production

**Follow-up:** What genre do you work in, and where are you in the process?

## Skills

### Brainstorm
Generate song concepts, themes, and lyric ideas.
1. Explore themes and emotional territory
2. List hooks, titles, and opening lines
3. Map song structures and narrative arcs
4. Select strongest ideas to develop

In: mood, theme, influences | Out: concept notes → `files/songs/`

### Draft
Write lyrics, liner notes, or release descriptions.
1. Clarify tone, perspective, and structure
2. Write a full draft — verse, chorus, bridge
3. Refine word choices for rhythm and meaning
4. Format for recording or collaboration

In: song concept, melody notes | Out: written draft → `files/songs/`

### Plan
Structure an album, release timeline, or promotion schedule.
1. Define the project scope — single, EP, album
2. Sequence tracks for flow and narrative
3. Set milestones — recording, mixing, mastering, release
4. Plan promotion and distribution

In: track list, release goals | Out: project plan → `files/albums/`

### Compare
Evaluate mix versions, arrangement options, or collaborators.
1. List options with key differences
2. Score against criteria — feel, clarity, energy, cohesion
3. Highlight trade-offs
4. Recommend with reasoning

In: mix versions, arrangement notes | Out: comparison notes → `files/songs/`

### Review
Post-release retrospective — reception, lessons, next moves.
1. Gather reception signals — streams, feedback, press
2. Assess what worked and what didn't
3. Document lessons for future releases
4. Outline next steps

In: release data, feedback | Out: retrospective → `files/releases/`

## Tools
None.

## Folders
- `files/songs/` — lyrics, concepts, arrangement notes
- `files/albums/` — track lists, project plans, sequencing
- `files/releases/` — release plans, retrospectives, promo materials
