# Home Renovation

**Follow-up:** What are you renovating, and what is the budget?

## Skills

### Research
Explore materials, costs, building codes, and contractor options.
1. Define what you need to know
2. Gather information — materials, regulations, pricing
3. Summarize options with pros, cons, and costs
4. Flag permit requirements or code constraints

In: project scope, photos of spaces, screenshots of design inspo | Out: research summary → `files/projects/`

### Plan
Define project scope, phases, timeline, and budget.
1. Break the project into phases
2. Estimate costs and duration per phase
3. Set milestones and decision points
4. Build a realistic schedule with buffer

In: project goals, research, budget | Out: project plan → `files/projects/`

### Compare
Evaluate contractors, materials, or designs side-by-side.
1. List options with key attributes
2. Score against criteria — cost, quality, timeline, reviews
3. Highlight trade-offs and hidden costs
4. Recommend with reasoning

In: PDF quotes, contractor proposals, material specs | Out: comparison table → `files/quotes/`

### Checklist
Generate task lists per project phase.
1. Break each phase into discrete tasks
2. Order by dependency and priority
3. Add owners, deadlines, and quality checks
4. Flag inspection or approval gates

In: project plan, current progress | Out: phase checklist → `files/projects/`

### Draft
Write project briefs for contractors or permit applications.
1. Clarify scope, standards, and expectations
2. Structure the document for the audience
3. Draft with precise, unambiguous language
4. Include relevant specs and references

In: project plan, requirements | Out: written brief → `files/projects/`
