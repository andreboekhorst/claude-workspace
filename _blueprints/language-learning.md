# Language Learning

**Follow-up:** What language are you learning, and where are you starting from?

## Skills

### Plan
Learning roadmap — goals, milestones, and daily routine.
1. Clarify target language, current level, and motivation
2. Set concrete milestones — what you can do at each stage
3. Design a daily practice routine with time estimates
4. Build a 4-week sprint plan to the first milestone

In: language, level, available time | Out: learning plan → `files/lessons/`

### Draft
Write practice dialogues and exercises.
1. Identify the grammar or vocabulary focus
2. Write a realistic dialogue using target structures
3. Add comprehension questions and fill-in exercises
4. Include an answer key with explanations

In: lesson topic, level | Out: practice exercises → `files/practice/`

### Teach
Grammar explanations and vocabulary lessons.
1. Introduce the concept with a clear, simple example
2. Explain the rule — when, why, and common exceptions
3. Provide a set of graded examples from easy to tricky
4. Contrast with the learner's native language where useful

In: grammar topic or vocabulary theme | Out: lesson → `files/lessons/`

### Review
Periodic progress check and goal adjustment.
1. Summarize what's been covered since last review
2. Test recall on key vocabulary and grammar
3. Identify weak spots and patterns in mistakes
4. Adjust the plan — accelerate, review, or redirect

In: completed exercises, lesson history | Out: progress review → `files/lessons/`

### Translate
Practice translations with feedback.
1. Present a passage in the source or target language
2. User provides their translation
3. Compare with a model translation — note differences
4. Explain errors and suggest natural alternatives

In: text to translate, user's attempt | Out: annotated feedback → `files/practice/`

## Tools
None required.

## Folders
- `files/lessons/` — learning plans, grammar lessons, progress reviews
- `files/vocabulary/` — word lists, flashcard sets, thematic groups
- `files/practice/` — dialogues, exercises, translation practice
