# Photography Project

**Follow-up:** What type of photography do you focus on?

## Skills

### Plan
Design a project — theme, locations, shot list.
1. Define the concept and visual story
2. Scout locations and plan logistics
3. Build a shot list with technical notes
4. Set a shooting schedule

In: project idea, reference images, mood boards | Out: project plan → `files/projects/`

### Research
Visual references, technique research, and location scouting.
1. Define what you need to learn or find
2. Gather references — styles, techniques, locations
3. Summarize findings with visual examples
4. Note practical takeaways for the shoot

In: project concept, photos of locations | Out: research notes → `files/references/`

### Brainstorm
Generate creative concepts and visual storytelling angles.
1. Explore the theme from multiple angles
2. List visual metaphors and narrative devices
3. Sketch compositions and sequences
4. Select strongest concepts to develop

In: theme, inspiration images | Out: concept list → `files/projects/`

### Critique
Portfolio review — strengths, gaps, style consistency.
1. Review the body of work as a whole
2. Identify strongest and weakest images
3. Assess style consistency and visual voice
4. Recommend additions, removals, or re-edits

In: portfolio images, artist statement | Out: critique notes → `files/portfolio/`

### Draft
Write artist statements or project descriptions.
1. Clarify audience and purpose
2. Distill the project's intent and meaning
3. Draft in the artist's voice
4. Edit for clarity and authenticity

In: project concept, critique notes | Out: written statement → `files/portfolio/`

## Tools
None.

## Folders
- `files/projects/` — project plans, shot lists, concepts
- `files/portfolio/` — portfolio selections, statements, critiques
- `files/references/` — mood boards, technique notes, visual research
