# Wedding Planning

**Follow-up:** When is the wedding, and what is the budget?

## Skills

### Plan
Build a full wedding timeline with milestones.
1. Set the date and work backwards
2. Map milestones — venue, vendors, invites, rehearsal
3. Assign deadlines to each milestone
4. Flag dependencies and decision points

In: wedding date, budget, priorities | Out: wedding timeline → `files/logistics/`

### Compare
Evaluate vendors, venues, or caterers side-by-side.
1. List options with key attributes
2. Score against criteria — cost, availability, style, reviews
3. Highlight trade-offs
4. Recommend with reasoning

In: vendor quotes, photos of venues, PDF contracts | Out: comparison table → `files/vendors/`

### Checklist
Generate week-by-week task lists.
1. Break timeline into weekly chunks
2. Assign actionable tasks per week
3. Add owners and deadlines
4. Mark critical-path items

In: wedding timeline, current progress | Out: weekly checklist → `files/logistics/`

### Draft
Write vows, speeches, or invitations.
1. Clarify tone, audience, and format
2. Gather key details and personal touches
3. Draft full text
4. Revise for flow and emotion

In: notes, anecdotes, tone preferences | Out: written draft → `files/personal/`

### Summarize
Create a day-of timeline pocket guide.
1. List every event in chronological order
2. Add times, locations, and responsible people
3. Flag transition moments and buffer time
4. Format for quick reference

In: full wedding plan, vendor schedules | Out: day-of guide → `files/logistics/`

## Tools
- **Google Calendar** — manage vendor meetings, deadlines, rehearsal schedule

## Folders
- `files/vendors/` — quotes, contracts, comparison notes
- `files/logistics/` — timelines, checklists, day-of guides
- `files/personal/` — vows, speeches, invitation drafts
